Welcome! This is a static HTML website that connects directly to your Redash API.

Quick Start:
  1. Verify your Redash URL, Query ID, and API Key above
  2. Set your desired date range
  3. Click "Fetch Data" to load your query results

Note: Due to browser security (CORS), this may only work if:
  • Your Redash instance allows cross-origin requests, OR
  • You open this file locally and your browser allows it, OR
  • You serve this file from the same domain as your Redash instance
